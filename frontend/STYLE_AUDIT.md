## Tailwind & PostCSS Versions

| Package | Version |
|---|---|
| `tailwindcss` | **4.1.10** |
| `@tailwindcss/postcss` | **4.1.10** |
| `postcss` | **8.5.5** |
| `@tailwindcss/typography` | **0.5.16** (devDependency) |

> **Tailwind v4 CSS-first mode** — no `tailwind.config.js` exists.  
> All design tokens live in the `@theme` block inside `src/styles.css`.

## PostCSS Configuration

File: `.postcssrc.json` (JSON format — **not** `postcss.config.js`)

```json
{
  "plugins": {
    "@tailwindcss/postcss": {}
  }
}
```

No autoprefixer, no cssnano, no additional transforms.

---

## Angular Build

`angular.json` styles array:
```json
"styles": ["src/styles.css"]
```
Angular 20 routes the CSS through PostCSS automatically — no custom builder override needed.

## `@layer` Setup in `styles.css`

### `@import "tailwindcss"` (line 1)
Single directive; Tailwind v4 injects utilities and resets from this import.

### `@layer base` (lines ~323–368)
- HTML/body reset (font, color, background via tokens)
- `focus-visible` ring (2px inner + 4px outer using `--color-ring`)
- Link reset
- Selection highlight using `--color-brand-soft`
- `@media (prefers-reduced-motion)` cut

### `@layer utilities` (lines ~373–454)
Semantic helper classes:
- `.bg-surface-0/1/2/3` — surface fills
- `.text-fg`, `.text-fg-muted`, `.text-fg-faint`
- `.border-border`, `.border-divider`
- `.lev-panel`, `.lev-raised` — elevation shadows
- `.ease-out`, `.dur-1`, `.dur-2` — motion
- `.h-control`, `.h-row`, `.p-widget`, `.gap-grid` — density-driven sizing
- `.mono-metric` — tabular-nums monospace block
- `.icon` — Material Symbols font-variation-settings
